# AI-Driven Code Review and Bug Detection Assistant

An intelligent multi-agent code review system built with LangChain, LangGraph, and Generative AI.

## Features
- 🐛 Bug Detection Agent
- 🔒 Security Audit Agent  
- ⚡ Performance Optimization Agent
- 🎨 Style & Best Practices Agent
- 📊 Weighted code quality scoring (0–100)
- 📄 JSON + Markdown reports with line-level comments

## Tech Stack
- **LangChain + LangGraph** — Multi-agent orchestration
- **Ollama / Gemini / Groq / OpenAI** — Pluggable LLM providers
- **FastAPI** — REST API
- **pylint, flake8, bandit, radon** — Static analysis
- **Python 3.10+**
